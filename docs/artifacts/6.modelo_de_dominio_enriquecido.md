## 1. Introducción

El **Modelo de Dominio Enriquecido** es el complemento del Modelo de Dominio Anémico (artefacto 5). Mientras el modelo anémico define los conceptos y sus relaciones superficiales, el modelo enriquecido profundiza en cada objeto de dominio con:

- **Caracterización completa de atributos**: tipo de dato, longitudes, formato, obligatoriedad, sensibilidad, autogeneración, si identifica el registro, valor por defecto, y comportamiento en cada operación (agregar, modificar, consultar por id, consultar lista)
- **Restricciones de unicidad** (combinaciones únicas de atributos que no deben repetirse)
- **Responsabilidades (métodos de negocio)**: cada operación documentada con entradas, salidas y políticas de negocio aplicables
- **Muestreo de datos reales**: ejemplos de datos representativos de cada entidad

El modelo cubre **33 objetos de dominio** organizados en los mismos dominios funcionales que el resto de la documentación SIBE.
